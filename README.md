1. Generate scaffold: `rails g scaffold Tweeet tweeet:text`
2. Add gems
    - bulma-rails
    - simple_form
    - better_errors
    - guard & guard-livereload

3. Work on layout with Bulma styles
4. Add devise user:
    - add User `rails g devise User`
    - add Registrations controller (manually)
    - add fields to User
